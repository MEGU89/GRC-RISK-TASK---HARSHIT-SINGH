   - Asset and Threat input fields
   - Likelihood and Impact sliders
   - Real-time score preview

## How to Capture Screenshots

1. Start both backend and frontend servers
2. Add sample data using the Add Risk form
3. Use browser dev tools (F12) to resize viewport if needed
4. Capture screenshots:
   - Windows: `Win + Shift + S`
   - macOS: `Cmd + Shift + 4`
   - Or use browser extensions like "Full Page Screen Capture"

Recommended dimensions: **1280x720** or **1920x1080**
